사용 설명서
📲 홈 화면에 추가하기
앱처럼 사용하려면 홈 화면에 추가하세요.
관리자 페이지와 계산기 모두 동일한 방법으로 추가합니다.
iOS (Safari에서)
1
Safari로 페이지 열기 (Chrome 불가)
2
하단 공유 버튼 □↑ 탭
3
홈 화면에 추가 선택 → 추가
🤖 Android (Chrome에서)
1
Chrome으로 페이지 열기
2
우측 상단 메뉴 ⋮ 탭
3
홈 화면에 추가 또는 앱 설치 선택
🔔 알림 설정
관객이 계산하면 관리자 기기로 푸시 알림이 옵니다.
1
홈 화면에서 앱으로 열기 iOS는 반드시 웹앱으로
2
메뉴 → 알림 받기 → 알림 구독 탭
3
권한 허용 → ✅ 완료 메시지 확인
iOS: 홈 화면 웹앱에서만 동작
🤖 Android: 브라우저에서도 동작
⚠️ 알림은 관리자 URL 기준으로만 전송됨
🔗 URL 생성 및 전달
관객에게 나눠줄 계산기 URL을 생성합니다.
1
메뉴 → URL 생성 탭
2
관객 기기에 맞는 URL 복사
iOS 또는 🤖 Android
3
카카오톡·문자 등으로 전달 → 관객이 홈 화면에 추가
⚠️ 관리자마다 고유 URL이 생성됩니다. 다른 관리자 URL과 섞이지 않도록 주의하세요.
👻 고스트 모드
관객 입력값을 관리자 폰에서 자동 재현합니다.
1
메뉴 → 고스트 모드에서 딜레이·속도 설정 후 저장
2
관객이 계산기에 숫자 입력 → 관리자 수신 내역에 표시
3
관리자 계산기에서 0 두 번 빠르게 탭
4
설정한 딜레이 후 버튼 애니메이션으로 결과값 자동 입력
🔢 자동 카운팅
관객 계산기가 자동으로 숫자를 셉니다.
1
관객 계산기를 세로로 세우면 N초 후 0부터 카운팅 시작
2
숫자 1~9 입력 후 = 누르면 그 숫자부터 카운팅
3
아무 버튼이나 누르면 중지 / 가로로 눕히면 해제
🔮 666 트릭
관객이 333+333을 입력하면 666이 나오고 자동 시퀀스가 실행됩니다.
1
관리자 계산기에서 = 세 번 빠르게 탭 → 666 모드 켜짐
2
관객이 333+333= 입력 → 666 표시 후 자동 시퀀스
3
취소하려면 = 두 번 빠르게 탭
타이밍은 메뉴 → 666 트릭에서 단계별로 조정 가능
🍎 포스 모드
관객이 어떤 계산을 해도 원하는 숫자가 결과로 표시됩니다.
1
메뉴 → 포스 모드에서 원하는 숫자 입력 후 저장
2
관객이 = 누르면 설정한 숫자가 결과로 표시
✨ % 버튼 자동 트리거
관객이 계산하면 관리자 화면에 계산식이 자동으로 나타납니다.
1
관리자 계산기에서 % 두 번 탭 → 버튼에 회색 점 표시 (대기)
2
관객이 계산 → 관리자 화면에 계산식 자동 실행
3
해제: % 한 번 탭 또는 % 두 번 탭
👁 수신 내역 미리보기
1
관리자 계산기에서 = 길게 누르기
2
최근 수신 내역 팝업 표시
전체 내역은 메뉴 → 수신 내역에서 확인
